Focus — positions-first
Row anatomy
Each row = ticker + position (strike · DTE · % DTE left) · G/L% · today's delta · progress toward the live target for that position's DTE bucket.
Target is dynamic
Your 60/60 rules change with % DTE left. Row target switches automatically:
>80% DTE left → 50%
41–79% → 60%
≤40% → 80%
Sort order
1. P1s (red strip)
2. P2s (amber strip)
3. Approaching target (74% of the way there)
4. Far from target

= your eye goes top-down for urgency.
Delta column
▲1.2 = +1.2pp since open today. The "did anything meaningful change" signal you were describing.
Filter chips
CSPs / CCs / LEAPs filter from your existing Open Positions tab — kept for familiarity.
Non-position alerts
Free-cash-below-floor stays as a banner above the list (dismissible, linkable). Not dominant.
